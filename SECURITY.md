# Security Policy

OpenZCine controls cameras over USB and local networks, so we take security seriously.

## Supported versions

The project is pre-release; only the latest `main` is supported. There are no released versions yet.

## Reporting a vulnerability

**Do not open a public issue for security vulnerabilities.**

Please report privately via [GitHub Security Advisories](https://github.com/erik-sutton95/OpenZCine/security/advisories/new). Include reproduction steps and impact if you can.

We aim to acknowledge reports within 7 days and to keep you updated as we investigate and fix the issue.
Coordinated disclosure is appreciated — we will credit reporters who wish to be named.
