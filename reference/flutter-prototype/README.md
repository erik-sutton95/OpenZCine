# Flutter Prototype Reference

This directory contains the archived Flutter prototype that proved early Nikon ZR pairing, PTP-IP,
live-view, and LUT workflow behavior.

Archived contents include the old `lib/`, `test/`, `shaders/`, and package metadata.

It is reference material only. The production app is native Swift/SwiftUI on iOS, with shared Swift
core code and a planned native Android shell. Root `just` recipes, CI, and the Xcode project do not
build or run this Flutter prototype.
